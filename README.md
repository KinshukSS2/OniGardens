# Oni Gardens Luxury Estate

A premium, immersive, single-page web application for a luxury mountain estate, designed to provide a high-end visual experience. 

![Oni Gardens Hero](src/assets/hero.png)

## 🌐 Live Demo
**[View the Live Site on Vercel](https://oni-gardens.vercel.app)**

## 🌟 Key Features
- **Cinematic Visuals:** High-resolution imagery with subtle, luxury-focused color palettes (Deep Forest & Moss Greens).
- **Smooth Animations:** Scroll-triggered entrance animations across all sections for a dynamic feel.
- **Responsive Layout:** Perfectly adapted for mobile, tablet, and desktop viewing.
- **Interactive Galleries:** Full-screen zoomable lightboxes for estate photography.

## 🛠️ Technology Stack
- **Frontend Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (with custom color variables)
- **Animations:** [GSAP](https://gsap.com/) (ScrollTrigger)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Gallery:** [LightGallery](https://www.lightgalleryjs.com/)

## 🚀 Local Development Setup

To run this project locally, make sure you have Node.js (v18+) installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KinshukSS2/OniGardens.git
   cd OniGardens
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy
The UI transitions away from generic white-space designs by leaning heavily into rich, saturated natural tones (`#0C2818` / `#0A1C11`). Typography and spacing are highly deliberate, utilizing serif fonts for headings to convey tradition and elegance, juxtaposed against clean sans-serif body text.
